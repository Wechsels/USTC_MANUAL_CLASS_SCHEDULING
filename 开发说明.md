# 开发说明

## 目录结构

```text
.
├── index.html              # 页面入口
├── js.js                   # 主要业务逻辑和排课算法
├── table.css               # 页面和课表样式
├── data.js                 # 生成后的课程数据
├── data_generator.py       # 从 class.xlsx 生成 data.js
├── class.xlsx              # 教务系统导出的课程数据
├── icourse_spider/         # 评课评分抓取和匹配
├── layui/                  # Layui 依赖
├── jquery.js               # jQuery 依赖
├── filesaver.js            # 导出文件依赖
├── README.md
└── LICENSE
```

## 本地运行

项目为静态页面，无构建步骤。可直接打开 `index.html`，或在项目根目录启动本地服务：

```bash
python -m http.server 8000
```

访问 `http://localhost:8000/`。

## 数据更新

1. 从 `https://catalog.ustc.edu.cn/query/lesson` 下载 Excel，并保存为根目录的 `class.xlsx`。
2. 按需修改 `data_generator.py` 末尾的 `semester`。
3. 安装依赖：

   ```bash
   python -m pip install pandas openpyxl
   ```

4. 生成课程数据：

   ```bash
   python data_generator.py
   ```

5. 检查 `data.js` 已更新，并在浏览器中验证搜索、添加课程和排课功能。

## 评分更新

通常不需要更新评分数据。如需刷新评课社区评分：

```bash
cd icourse_spider
python -m pip install requests tqdm
python spider.py
cd ..
python data_generator.py
```

## 开发流程

- 页面结构改 `index.html`。
- 业务逻辑和排课算法改 `js.js`。
- 样式改 `table.css`。
- 课程数据生成逻辑改 `data_generator.py`。
- 不建议手动修改 `data.js`，该文件会被生成脚本覆盖。
